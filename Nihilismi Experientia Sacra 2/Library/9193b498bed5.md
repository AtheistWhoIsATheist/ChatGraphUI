# Sacred Dread Texture Analyzer

Analyze philosophical texts for Nihiltheistic texture—the felt-quality of void-experience through negation, dread, and sacred resonance.

## Overview

This skill provides systematic texture analysis for philosophical content, drawing from the Journal314 unified voice dataset (3,893 quotes across 52 thinkers). It classifies text by voice markers, maps phenomenological qualities, calculates void-resonance scores, and identifies transformation potential within the Nihiltheistic framework.

**Core Distinction:**
- **Texture**: The felt-quality of writing—how experience is conveyed, not just what is said
- **Voice**: The philosophical persona underlying the text (cynic, mystic, ascetic, nihilist)
- **Resonance**: Cross-thinker kinship at the level of operator and disposition

**When to Use:**
- Analyzing quotes or passages for nihiltheistic character
- Comparing voice/texture across different thinkers
- Assessing whether a passage refracts the Transcendent through negation
- Evaluating existential-mystical writing samples
- Generating thinker profiles based on text samples

### Step 2: Texture Extraction

Analyze text across dimensions:

```python
def analyze_texture(text: str) -> dict:
    """Calculate texture dimensions for input text."""
    void_markers = [
        'void', 'nothingness', 'groundless', 'empty', 'absence',
        'collapse', 'dissolve', 'vanish', 'cease', 'annihilate'
    ]
    dread_markers = [
        'dread', 'terror', 'anguish', 'horror', 'fear', 'uncanny',
        'unheimlich', 'anxious', 'paralyzed', 'vertigo'
    ]
    ascetic_markers = [
        'renounce', 'detach', 'strip', 'empty', 'purge', 'mortify',
        'kenosis', 'self-emptying', 'surrender', 'let go'
    ]
    mystical_markers = [
        'union', 'silence', 'presence', 'stillness', 'dark', 'cloud',
        'beyond', 'ineffable', 'unsayable', 'fullness'
    ]
    
    void_score = sum(1 for m in void_markers if m in text.lower()) / len(void_markers)
    dread_score = sum(1 for m in dread_markers if m in text.lower()) / len(dread_markers)
    ascetic_score = sum(1 for m in ascetic_markers if m in text.lower()) / len(ascetic_markers)
    mystical_score = sum(1 for m in mystical_markers if m in text.lower()) / len(mystical_markers)
    
    return {
        'void_contact': round(void_score, 3),
        'dread_saturation': round(dread_score, 3),
        'ascetic_renunciation': round(ascetic_score, 3),
        'mystical_resonance': round(mystical_score, 3),
        'transcendent_echo': calculate_transcendent_echo(void_score, mystical_score)
    }
```

### Step 3: Voice Classification

Determine dominant voice:

```python
def classify_voice(text: str, texture: dict) -> str:
    """Classify philosophical voice based on texture."""
    if texture['ascetic_renunciation'] > 0.3:
        return 'cynic-ascetic'
    elif texture['mystical_resonance'] > 0.3 and texture['void_contact'] > 0.2:
        return 'mystical-apophatic'
    elif texture['dread_saturation'] > 0.4:
        return 'existential-dread'
    elif texture['void_contact'] > 0.4:
        return 'pessimist-nihilist'
    elif texture['transcendent_echo'] > 0.5:
        return 'transcendentalist'
    else:
        return 'mixed-generic'
```

### Step 4: Groundlessness Flag

Set groundlessness flag if void-contact is dominant:

```python
def check_groundlessness(text: str, texture: dict) -> bool:
    """Determine if text exhibits groundlessness signature."""
    groundlessness_patterns = [
        'ground collapses', 'no foundation', 'groundless ground',
        'abyss beneath', 'void beneath', 'everything dissolves',
        'i cease', 'self dissolves', 'no self', 'ultimate groundlessness'
    ]
    
    pattern_matches = sum(1 for p in groundlessness_patterns if p in text.lower())
    return texture['void_contact'] > 0.4 and pattern_matches >= 1
```
