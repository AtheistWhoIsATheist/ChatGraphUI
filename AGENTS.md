# AGENTS.md - Persistent Project Intelligence

## Project Context: Journal314 & REN Analysis Engine
An advanced structural textual analysis platform for deconstructing philosophical reports (Journal314) and mapping them onto Recursive Existential Nihilism (REN) topologies.

## Core Directives (Context Engineering)
- **High Intentionality**: All UI must be distinctive, avoiding generic patterns. Use the Antigravity Design system.
- **Recursive Self-Improvement**: Log all optimizations in `/src/utils/selfImprovement.ts`.
- **AxiomForge (Semantic Oracle)**: 100% robust extraction of nodes and links from raw text substrate.
- **Source-Driven Development (SDD)**: Ground every implementation in official documentation. Treat documentation as the "Ground Truth" for technical substrate. Cite sources via code comments.
- **Agnostic Topology**: Maintain strict agnosticism toward "Truth" in the ontology. Frame all nodes as "Phenomenological Occurrences" or "Possibilities".

## Tech Stack
- Frontend: React 18 (Vite), TypeScript, Tailwind CSS 4, Motion (for animations).
- Visualization: D3.js (2D Graph), Three.js (3D Topology).
- AI: Gemini 1.5 Pro / 2.0 Flash via @google/genai.
- Icons: Lucide-React.

## Skills & Knowledge Layers
- **Real-Time Data Extraction**: Through `AISynthesisPanel`.
- **Structural Integrity**: Automated node/link validation.
- **Topological Auditing**: Maintenance of a cryptographic audit trail for all syntheses.

## Directory Structure
- `/src/components`: Specialized UI components (AxiomForge, Graph, Sidebar).
- `/src/data`: Nihiltheism (NT) Ontology and corpus data (`corpus.ts`, `nt_schema.ts`).
- `/src/utils`: Utilities for ingestion and self-improvement tracking.

## Development Workflow
- **Lint**: `npm run lint`
- **Build**: `npm run build`
- **Verification**: `npm run compile_applet`
